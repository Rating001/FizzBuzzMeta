Fizz Buzz App

Print the numbers 1 through 100, but if the number is divisible by #A then print Fizz, if the number is divisible by #B then print Buzz; if the number is divisible by both #A and #B then print FizzBuzz.

Instructions

  1. Enter a number above 0 for Fizz and Buzz.
  2. Change the way FizzBuzz is solved if you'd like.
  3. Choose whether you want color added to the output.
  4. Press "Fizz Buzz!"
  5. Make many "Ooooooo and Ahhhhhhhs"

Note: Numbers will be rounded down to the next whole number.
